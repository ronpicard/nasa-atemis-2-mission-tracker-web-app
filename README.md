# Artemis II Mission Tracker (web app)

React + TypeScript + Vite dashboard: clocks, 3D trajectory, telemetry, NASA RSS wire, timeline replay, and links to official coverage. Not affiliated with NASA.

- **Live (GitHub Pages):** after `npm run deploy`, the site is served from the `gh-pages` branch.
- **Official tracking:** [nasa.gov/trackartemis](https://www.nasa.gov/trackartemis)

## Scripts

```bash
npm install
npm run dev      # local development
npm run build    # production build
npm run deploy   # build for GitHub Pages + push to gh-pages branch
```
